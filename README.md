# incident-response-labs

This repository is used to document incident response learning, labs, and practice exercises.

## Structure

- **notes/** – Theory notes, study material, and references
- **incident-reports/** – Write-ups of simulated or real-world incident scenarios
- **scripts/** – Detection, analysis, or automation scripts
- **screenshots/** – Evidence and visual artifacts from investigations

## Purpose

Hands-on practice for Security Operations and Incident Handling.
